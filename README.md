# maven-dependency-version-updates
## Sample Command
```
java -jar maven-dependency-version-updates-0.0.1-SNAPSHOT.jar file=pom.xml groupId=org.apache.commons artifactId=commons-lang3 version=3.20.0
```
## Sample Output
```
groupId=org.apache.commons,artifactId=commons-lang3,version=[3.19.0->3.20.0]
```
